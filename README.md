# How to Cheat aka Add Points

1. Right Click and press "Inspect element".
2. Go to Console
3. type "clicks = HOW_MANY_YOU_WANT"
4. Replace HOW_MANY_YOU_WANT with number of how much you want. If you want 300 then type clicks = 300
